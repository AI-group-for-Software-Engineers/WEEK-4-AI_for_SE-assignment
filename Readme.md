# Task 1: AI-Powered Code Completion

## Description
This project compares AI-generated code (GitHub Copilot style) with manual implementation for sorting a list of dictionaries by a specific key.

## Files
- `sample_data.py` — Generates sample test data.
- `ai_impl.py` — AI-based implementation.
- `manual_impl.py` — Manual version.
- `tests.py` — Functional correctness tests.
- `benchmarks.py` — Performance and memory benchmark.
